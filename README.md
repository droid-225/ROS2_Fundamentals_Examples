# ROS 2 Fundamentals Examples

This package contains examples demonstrating fundamental ROS 2 concepts

## Description

This package includes minimalist ROS 2 code to demonstrate important ROS 2 concepts and patterns.
- Following ROS 2 Python style guidelines

## Prerequisites

- ROS 2 installed
- Python 3
- Created a ROS 2 workspace (`ros2_ws`)

## Notes

- Video wise notes are stored in the notes.txt file inside the inner ros2_fundamentals_example folder for reference.
- The build folder was not added to the git, make sure to have that configured that when trying the code.
